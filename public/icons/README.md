# Icons Folder

This folder contains all favicon and app icon files for the HyperFabrics website.

## Required Files

- `favicon.ico` - Main browser favicon (32x32 or 16x16)
- `favicon-16x16.png` - Small favicon
- `favicon-32x32.png` - Standard favicon
- `apple-touch-icon.png` - iOS home screen icon (180x180)
- `icon-192.png` - PWA icon for Android (192x192)
- `icon-512.png` - PWA icon for Android (512x512)

## How to Generate

Use one of these tools:

- https://realfavicongenerator.net/ (Recommended)
- https://favicon.io/
- Design in Figma/Canva and export

## File Specifications

| File                 | Size         | Format | Purpose           |
| -------------------- | ------------ | ------ | ----------------- |
| favicon.ico          | 16x16, 32x32 | ICO    | Browser tab       |
| favicon-16x16.png    | 16x16        | PNG    | Small displays    |
| favicon-32x32.png    | 32x32        | PNG    | Standard displays |
| apple-touch-icon.png | 180x180      | PNG    | iOS home screen   |
| icon-192.png         | 192x192      | PNG    | Android PWA       |
| icon-512.png         | 512x512      | PNG    | Android PWA       |

## Design Tips

- Use your brand colors
- Keep design simple and recognizable at small sizes
- Test at different sizes
- Ensure good contrast
- Include your logo or brand mark
